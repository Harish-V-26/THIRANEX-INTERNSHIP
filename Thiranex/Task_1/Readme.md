# 🔐 Password Sentinel

> Built as part of my internship at **Thiranex**

A real-time password strength analyzer that evaluates security across 10 dimensions, estimates crack time, and suggests stronger alternatives — all client-side with zero data transmission.

## Features
- ✅ 10-point security check (length, complexity, patterns, common passwords)
- ⚡ Entropy & crack-time estimation (10B guesses/sec model)
- 🔑 SHA-256 hashing for password history & reuse detection
- 💡 Smart hardened suggestions + random 16-char alternatives
- 🛡️ Fully client-side — no data ever leaves your browser

## Tech Stack
React · Web Crypto API · Client-side Security
